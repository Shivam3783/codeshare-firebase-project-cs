
## Getting Started


Installation of Redis on Mac:
```bash
➤➤ brew install redis
➤➤ brew services start redis
➤➤ brew install redisinsight
```

```bash

➤➤ brew services start redis
➤➤ npm install
➤➤ npm run dev
➤➤ npm run dev -- -H 0.0.0.0       //for network 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
